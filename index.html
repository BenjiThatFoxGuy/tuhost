Welcome to the TUHost Repository

This page hosts files used for loading images, custom webpages, and media into my Tower Unite condos. If you're seeing this in Tower Unite, it means something went wrong or you typed the URL incorrectly.

What is Tower Unite?

Tower Unite is a multiplayer virtual world that allows you to load custom media, websites, and images into your condo using a built-in Chromium browser. This repository is a Tower Unite fork of my original VRChat host, adapted for the features available in Tower Unite. While it allows loading images and custom media, string loading is not supported in Tower Unite like it is in VRChat.

Repository Purpose

This repository houses public files specifically for loading images, custom webpages, and media into my Tower Unite condos. It is not open for external contributions such as issues, pull requests, or discussions.
